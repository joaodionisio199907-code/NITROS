Preguntas Frecuentes

Will I get banned from GTA Online?

No. NitroSMP doesn't interact with Rockstar Online Services. It uses its own hooks and doesn't modify your original game files.

Does it work with Epic Games / Steam?

Yes, NitroSMP is compatible with all official legal versions of Grand Theft Auto V.

Can I create my own server?

Absolutely. We provide the server files for free. You can script in C#, JavaScript or Lua.
